# LakSeva - Local Service Marketplace

A modern Angular-based platform connecting people with local service providers in Sri Lanka.

## Overview

LakSeva is a location-based service marketplace designed for Sri Lankan communities. The platform helps users discover and book services from nearby providers across the country, with support for multiple languages and local payment methods.

## Tech Stack

- Frontend: Angular
- Styling: CSS/SCSS
- Multi-language support (Sinhala, Tamil, English)

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn
- Angular CLI

### Installation

```bash
# Install Angular CLI globally
npm install -g @angular/cli

# Install dependencies
npm install

# Run development server
ng serve
```
