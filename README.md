## Overview

**LakSeva** connects everyday people with trusted local service providers across Sri Lanka — from home repairs and personal care to tutoring and catering. Built for the community, by the community.

---

## Features

- Search & Discover — find any local service by category or location
- Location-based browsing — providers near you across Sri Lanka
- Ratings & Reviews — make informed decisions with community feedback
- Multi-language support — Sinhala, Tamil, and English
- Mobile friendly — fully responsive across all devices
- Provider profiles — detailed profiles for every service provider

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Scripts

| Command | Description |
|---------|-------------|
