# LakSeva - Sri Lanka's Local Service Marketplace

LakSeva is a community-focused platform that brings together all local services available across Sri Lanka — from household repairs and personal care to professional and trade services. The goal is to make it easy for everyday people to find and connect with trusted local service providers in their area.

## Vision

To be the go-to platform for discovering and accessing every type of local service in Sri Lanka, empowering local communities and small service providers across the country.

## What It Covers

- All categories of local services available in Sri Lanka
- Service providers at the community level (not large corporations)
- Location-based discovery — find services near you
- Multi-language support: Sinhala, Tamil, and English

## Tech Stack

- Framework: Next.js (App Router)
- Language: TypeScript
- Styling: Tailwind CSS

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start development server |
| `npm run build` | Build for production |
| `npm run start` | Start production server |
| `npm run lint` | Run ESLint |
